🔴 Critical (Phải có) Không được bỏ qua
▶
  • JWT secret ≥ 32 ký tự, random, không commit vào Git. Kiểm tra: grep -r "jwt.secret" trong source nếu thấy value = hard fail
  • DevDataSeeder bị disable ở production: @Profile("dev") annotation hoặc env var guard. Chạy seed trên prod = data breach risk
  • GET /exams/{id}/take không trả đáp án đúng: response DTO không có correctAnswer, isCorrect. Verify bằng DevTools Network tab
  • CORS chỉ allow production domain: không còn localhost:3000 hay wildcard * trong prod config
  • Spring Security: /admin/**, /teacher/** bị guard đúng role: test manual với token student
  • Password BCrypt trong DB: SELECT password FROM users LIMIT 1 phải bắt đầu bằng $2a$
  • Rate limiting trên auth endpoints: /login, /forgot-password tối đa 5 req/min per IP
  • HTTPS enforced: HTTP → HTTPS redirect hoạt động
  • Environment variables không log ra console: không có System.out.println(apiKey) hoặc console.log(token) trong prod build
  • DB credentials trong env var, không hardcode trong application.properties. File .env không được commit
  • IDOR check trên tất cả endpoint có ID trong path (/progress/user/{id}, /mistakes/{id}, /exams/{id}/results)
  • SQL injection protection: tất cả query dùng JPA/PreparedStatement, không string concatenation
🟡 High (Nên có) Ảnh hưởng nghiêm trọng nếu thiếu
▶
  • Quest reward idempotent: completeQuest được gọi 2 lần chỉ cộng XP 1 lần
  • ExamResult idempotent: submit 2 lần không tạo 2 records
  • completeLesson idempotent: gọi nhiều lần không tăng stats nhiều lần
  • Timezone nhất quán: server và DB cùng UTC hoặc cùng +07:00. Daily quest reset đúng 00:00 HCM time
  • DB unique constraints: email, (userId+examId), (userId+lessonId), (userId+questId+date)
  • Pagination trên tất cả list endpoint: không có endpoint nào trả unlimited records
  • Global error handler: @ControllerAdvice trả JSON error, không bao giờ trả HTML stack trace
  • Axios interceptor hoạt động: 401 → auto refresh → retry original request
  • Empty states: mọi trang list đều handle empty array gracefully
  • Loading states: mọi API call đều có skeleton/spinner
  • Vocabulary review XP không farm được: 1 từ / 1 ngày / 1 user
🟢 Medium (Nên cải thiện)
▶
  • Mobile responsive: test trên 375px (iPhone SE), 390px (iPhone 14), 768px (iPad)
  • Bundle size: initial JS < 500KB, lazy load các route ít dùng
  • Images: alt text đầy đủ, lazy loading, webp format
  • Lighthouse score: Performance ≥ 80, Accessibility ≥ 85
  • 404 page: custom page với link về dashboard
  • Console.log và debug code được xóa trong prod build
  • Favicon, og:image, title tag đúng cho từng trang
  • API response time < 500ms cho các endpoint thường dùng
